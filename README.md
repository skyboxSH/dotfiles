# My Dotfiles!

You will need:
- **hyprland** (window manager)
- **hyprshot** (screenshot utility)
- **hyprpaper** (wallpaper utility)
- **waybar** (status bar)
- **otf-font-awesome** (Icons)
- **dunst** (notification daemon)
- **pavucontrol** (audio mixer)
- **nm-applet** (network manager)
- **wofi** (application launcher)
